# Ping-pong

Простая игра в пинг-понг, реализованная на Python с использованием библиотеки PyGame.

## Описание игры

Это классическая игра в пинг-понг для двух игроков. Цель игры - отбивать мяч ракеткой и не позволить ему уйти за пределы вашей стороны поля.

## Управление

### Игрок слева:
- **W** - движение ракетки вверх
- **S** - движение ракетки вниз

### Игрок справа:
- **Стрелка вверх** - движение ракетки вверх
- **Стрелка вниз** - движение ракетки вниз

## Правила игры

- Мяч отскакивает от верхней и нижней границ поля
- При столкновении с ракеткой мяч меняет горизонтальное направление
- Игра заканчивается, когда мяч уходит за левую или правую границу поля
- Побеждает игрок, на чьей стороне мяч не оказался за границей

## Особенности реализации

- Размер окна: 900x800 пикселей
- Синий фон (RGB: 135, 206, 250)
- Частота обновления экрана: 60 FPS
- При отскоке от ракетки мяч меняет только горизонтальное направление
- Ракетки не могут выходить за границы игрового поля

## Запуск игры

Для запуска игры необходимо:
1. Установить Python
2. Установить библиотеку PyGame: `pip install pygame`
3. Запустить файл `main.py`

## Файлы проекта

- `main.py` - основной файл игры
- `player-left.png` - изображение левой ракетки
- `player-right.png` - изображение правой ракетки  
- `ball.png` - изображение мяча

## Технические детали

Игра использует:
- Класс `GameSprite` как базовый класс для всех игровых объектов
- Класс `Platform` для управления ракетками
- Класс `Ball` для управления поведением мяча
- Систему обработки столкновений через `colliderect()`
- Отображение победных сообщений при завершении игры
